What I've Built

👁️ Computer Vision / AI Wearable Almost Done

See For You

An AI-powered wearable assistive system in the form of smart glasses for visually impaired individuals. Built on a Raspberry Pi Zero 2 W with a stereo camera module, it applies real-time object detection (YOLOv8-tiny) and depth estimation to identify obstacles like pedestrians, vehicles, and stairs — all processed on-device via edge inference. Environmental information is communicated through directional haptic feedback (vibration motors) and audio alerts via Bluetooth smart glasses, without blocking ambient sounds. A motion-triggered OCR mode recognizes short text such as medicine labels, currency notes, and signage, converting it to speech. A low-cost, infrastructure-free wearable that combines object detection, stereo depth perception, OCR, and intelligent feedback to enhance mobility and independence.
